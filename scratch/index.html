Top Places to Run


  1. Chamonix, France
    • Stunning views of Mont Blanc
    • Insane trails and terrain
    • One of the biggest trail running hubs
  2. Wengen, Switzerland
    • Incredible alpine views
    • Run straight from town
    • Only accessible by train, not as crowded
  3. Engelberg, Switzerland
    • Running through herds of cows
    • Run along crystal blue mountain lakes
    • Completely open and so green
  4. Mendoza, Argentina
    • Running in the Andes
    • Running through vineyards is a unique experience
    • Served my mission here
  5. Salt Lake City, Utah
    • Great views of the Wasatch mountains
    • Great trails and running paths
    • Close to BYU
  6. Jackson, Wyoming
    • Beautiful views of the Tetons
    • Everywhere is accessbile by trail
    • Chance to see a bear
